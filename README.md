 Gestor de Hidratação Terapêutico
Este projeto é uma aplicação web focada em saúde e bem-estar, projetada para auxiliar usuários no controle de sua hidratação diária. A interface prioriza a experiência do usuário (UX) e a responsividade, garantindo que o monitoramento seja simples e intuitivo em qualquer dispositivo.

Objetivo do Projeto
Criar uma ferramenta prática onde o usuário possa gerenciar ciclos de tempo, visualizar seu progresso diário através de um dashboard e manter um registro histórico de suas atividades.

Tecnologias e Conceitos
HTML5 & CSS3: Arquitetura semântica e estilização moderna com Flexbox.

JavaScript (Vanilla): Lógica de temporização, gerenciamento de estado e manipulação dinâmica do DOM.

Responsividade: Uso de Media Queries para adaptação em dispositivos móveis.

UX/UI Design: Foco em hierarquia visual e feedback imediato ao usuário (alertas sonoros e visuais).

 Funcionalidades Principais
Dashboard Informativo: Exibição em tempo real do total de ciclos concluídos e horário da última atualização.

Cronômetro de Ciclos: Timer interativo para controle de intervalos.

Histórico de Atividade: Registro automático de cada ciclo concluído, permitindo auditoria visual das tarefas.

Reset de Dados: Função para limpar o histórico e reiniciar o contador do dia.

Engenharia de Software aplicada
Este projeto demonstra competências fundamentais de desenvolvimento frontend:

Clean Code: Organização de funções por responsabilidade.

Event Handling: Captura e tratamento de eventos de clique e tempo.

DOM Manipulation: Atualização da interface sem necessidade de recarregar a página.
