Gestor de Hidratação Terapêutico

Descrição: Aplicação focada em bem-estar que utiliza ciclos de tempo personalizados para auxiliar na hidratação diária. O projeto une lógica de programação em JavaScript puro com uma interface intuitiva e autoral.

Status:Em desenvolvimento (Refinando funcionalidades de áudio e persistência).

Tecnologias: HTML5, CSS3 e JavaScript (Manipulação de DOM e Intervalos).
